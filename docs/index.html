Sonara

Ver Demo Código Fuente

Sobre el Proyecto

Una nueva forma de interactuar con tus artistas favoritos.

Sonara es una aplicación web full-stack que permite a los usuarios buscar canciones, guardarlas como favoritas y explorar nueva música. Fue desarrollada con un fuerte enfoque en la experiencia de usuario (UX/UI), aprovechando el poder de la API oficial de Spotify.

Stack Tecnológico

React Node.js Express PostgreSQL Spotify API HTML5 CSS3 JavaScript
